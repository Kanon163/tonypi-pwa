# TonyPi ADHD PWA Demo

本工作区用于把既有 TonyPi ADHD 干预游戏 demo，重构为安卓手机可用的 PWA 产品 demo。

产品 SKU：BCI 头环、TonyPi 机器人、手机 PWA。

## 入口文档

- `AGENTS.md`：所有 agent 的硬约束
- `docs/PROJECT_BRIEF.md`：项目目标和参考资料
- `docs/WORKFLOW.md`：阶段、优先级、交付节奏
- `docs/CONTRACTS.md`：跨 agent 共享契约
- `docs/GITHUB_WORKFLOW.md`：GitHub Issue、PR、Pages 协作规则
- `docs/AGENT_BRIEFS.md`：子工作区分工
- `tasks/TEMPLATE.md`：开任务模板
