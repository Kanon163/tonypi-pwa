# 风险登记

## 高风险

- BCI 设备协议未知，可能影响真实接入时间。
- 手机与 TonyPi 网络连接方式会影响交付稳定性。
- 旧 demo 资源存在编码与耦合问题。

## 中风险

- agent 并行产出可能互相不兼容。
- 家长端 PWA 与儿童端机器人体验可能目标错位。
- 云端报告若过早复杂化，会拖慢垂直切片。
- Wave 1 产物若不压缩为契约和 fixtures，会变成阅读负担。
- 真实 `AGC.runActionGroup` 的阻塞、抢占、停止语义未验证。
- `BGM03` 被旧 demo 引用但资源目录未发现。
- Wave 3 mock 闭环若被误解为真实硬件能力，会影响演示边界。

## 应对

- 先使用模拟 BCI 和模拟机器人桥完成垂直切片。
- 通过 `CONTRACTS.md` 管控跨模块接口。
- 通过 `PROJECT_STATE.md` 做每波同步。
- Wave 2 先冻结契约和 fixtures，再进入 PWA 实现。
- Wave 3 页面和报告必须标注模拟或降级状态。
