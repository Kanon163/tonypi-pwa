# 产品愿景

## 产品定义

本项目目标是交付产品级 PWA demo。产品 SKU 包含：

- BCI 头环：采集脑电数据，分析注意力相关指标。
- TonyPi 机器人：儿童训练中的主要具身交互对象。
- 手机 PWA：家长的服务入口、训练启动器、训练管理和报告界面。

## 用户

- 家长：开启训练、连接设备、查看训练过程摘要和报告。
- 儿童：佩戴 BCI 头环，与 TonyPi 机器人完成执行功能训练。
- 演示/服务人员：配置设备、排查连接、辅助完成演示。

## 核心体验

家长通过 PWA 开启训练；儿童佩戴 BCI 头环，主要与 TonyPi 互动；训练数据被记录并在 PWA 中形成家长可理解的反馈。

## 产品判断

- PWA 不是旧 demo 的控制台迁移版。
- 儿童不应主要盯着手机训练。
- TonyPi 不是外设，而是训练体验的核心角色。
- BCI 数据不是装饰性指标，必须进入训练记录或反馈策略。
- 报告必须服务家长理解训练，而不是堆叠原始数据。

## 首版成功标准

- 家长能在 PWA 中完成设备连接和训练开启。
- 儿童能跟 TonyPi 完成至少一段完整训练。
- BCI 注意力指标能进入训练记录。
- PWA 能展示训练摘要或报告雏形。
- 任一设备异常时有可解释的降级路径。
